How It Works

One model. One endpoint. All orchestration invisible.

# Install pip install -r requirements.txt # Start the API python -m litellm --config config/litellm.yaml --port 4000 # Use it (OpenAI-compatible) import openai client = openai.OpenAI(base_url="http://localhost:4000/v1", api_key="your-key") response = client.chat.completions.create( model="temuclaude", messages=[{"role": "user", "content": "What is 15*12?"}] ) print(response.choices[0].message.content) # "180"
